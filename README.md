 Features

- Calculate CGPA instantly
- Add multiple subjects dynamically
- Enter grades and credit values
- Real-time calculation
- Responsive design for desktop and mobile
- Clean and user-friendly interface

Tech Stack
- HTML5
- CSS3
- JavaScript

Project Structure

cgpa-calculator/
│── index.html
│── style.css
│── script.js
│── README.md

 Installation

1. Clone the repository
git clone https://github.com/sadhusravya123/cgpa-calculator.git

2. Navigate to the project folder
cd cgpa-calculator

3. Open index.html in your browser.
No additional dependencies or installation required.

How to Use
1. Enter the grade points for each subject.
2. Enter the corresponding credit values.
3. Add more subjects if needed.
4. Click Calculate.
5. View your CGPA instantly.
